# AWS Distro for OpenTelemetry (ADOT)

AWS Distro for OpenTelemetry (ADOT) is a secure, AWS-supported distribution of the cloud-native OpenTelemetry project.

## Managed Open-Source Observability

### Vendor-Agnostic Instrumentation
ADOT provides a standardized way to implement vendor-agnostic instrumentation. It is used to pull traces and metrics out of your applications without locking your code into a specific vendor's proprietary SDK.

### Data Routing
Once ADOT collects the telemetry data from your workloads (like EKS or ECS), it can route that data to multiple destinations simultaneously, such as Amazon CloudWatch, AWS X-Ray, or Amazon Managed Prometheus, giving you ultimate flexibility in your observability stack.
