# Amazon Managed Grafana (AMG)

Amazon Managed Grafana (AMG) is a fully managed visualization platform that allows you to instantly query, correlate, and visualize operational metrics, logs, and traces.

## Managed Open-Source Observability

### Centralized Dashboards
AMG connects cleanly to multiple data sources—including Amazon Managed Prometheus (AMP), CloudWatch, AWS X-Ray, Amazon OpenSearch, or third-party plugins—allowing you to spin up custom, interactive operational dashboards in a single pane of glass.

### Operational Efficiency
Like other managed open-source offerings, it removes the heavy lifting of provisioning servers, configuring software, and scaling infrastructure, while providing built-in security integrations like AWS IAM Identity Center (SSO) for user access.
