# Amazon Managed Service for Prometheus (AMP)

Amazon Managed Service for Prometheus (AMP) is a highly available, secure, and fully managed scratch space container monitoring environment designed to scrape and evaluate PromQL metrics at scale.

## Managed Open-Source Observability

### Purpose-Built for Containers
AMP is designed for engineering teams utilizing cloud-native CNCF open-source tooling (like Kubernetes), eliminating the operational overhead of managing underlying storage, patching, or scaling compute nodes for Prometheus servers.

### Integration
It seamlessly receives metrics scraped by AWS Distro for OpenTelemetry (ADOT) or native Prometheus servers running in Amazon EKS, and naturally pairs with Amazon Managed Grafana for visualization.
