# Amazon FSx

Amazon FSx makes it easy and cost-effective to launch and run popular, fully managed, highly performant commercial and open-source file systems.

## 1. Specialized Commercial & Open-Source File Deployment

### FSx for Windows File Server
Provides fully managed, highly reliable, and scalable file storage accessible over the industry-standard SMB protocol. It features full native integration with Microsoft Active Directory (both AWS Managed AD and on-premises AD) and supports Windows NTFS features.

### FSx for Lustre
A high-performance, sub-millisecond latency file system optimized for fast processing of workloads like High Performance Computing (HPC), machine learning, and video processing. It can be linked directly to Amazon S3 data-lakes, seamlessly presenting S3 objects as files.

### FSx for NetApp ONTAP
Provides multi-protocol access (NFS, SMB, and iSCSI) from Linux, Windows, and macOS clients. It brings enterprise-grade ONTAP features to the cloud, including highly efficient snapshots, cloning, data compression, and deduplication.

### FSx for OpenZFS
Delivers high-throughput, ZFS-powered file systems built for rapid migration of commodity Linux servers. It provides the familiar features of OpenZFS (like ZFS snapshots and cloning) with the agility and scalability of AWS.
