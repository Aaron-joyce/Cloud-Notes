# AWS Audit Manager

AWS Audit Manager is an automated compliance assessment tool that maps your AWS usage data directly to standard regulatory frameworks.

## Compliance Tracking & Resource Auditing

### Automated Evidence Collection
Audit Manager simplifies how you assess risk and compliance with regulations and industry standards (such as PCI-DSS, HIPAA, GDPR, and SOC 2). It continuously collects evidence from multiple AWS sources:
- AWS CloudTrail (User activity)
- AWS Config (Resource configurations)
- AWS Security Hub (Security findings)

### Audit-Ready Reports
Instead of manually collecting evidence and mapping it to specific compliance controls, Audit Manager automates this mapping and generates audit-ready compliance reports, significantly reducing the manual effort required for audits.
