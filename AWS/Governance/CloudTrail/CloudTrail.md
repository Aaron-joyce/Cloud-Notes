# AWS CloudTrail

AWS CloudTrail is the definitive operational governance service that captures a comprehensive audit trail of API actions within your accounts.

## Compliance Tracking & Resource Auditing

### Operational Governance
CloudTrail records all API activity across your AWS infrastructure, providing visibility into user and resource activity.

### Audit Logging
It logs critical metadata about every API call:
- **Who**: The IAM user or role that made the call.
- **When**: The exact timestamp of the request.
- **Where**: The source IP address of the caller.
- **What**: The parameters that were passed and the response returned.

### Use Cases
CloudTrail is essential for:
- **Forensic Security**: Investigating unauthorized access or security incidents.
- **Regulatory Reporting**: Providing an immutable record of system changes to auditors.
- **Troubleshooting**: Discovering exactly when a resource was modified or deleted.
