# Amazon Translate

Amazon Translate is a neural machine translation service that delivers fast, high-quality, affordable, and customizable language translation.

## Cross-Lingual Translation Features

### Neural Machine Translation
It uses deep learning models to deliver more accurate and natural-sounding translation than older statistical translation algorithms. It fluently translates vast amounts of text for localization or real-time communication.

### Advanced Capabilities
- **Language Tracking / Auto-Detection**: If the source language is unknown, Translate can automatically identify it before performing the translation.
- **Custom Terminology**: Allows organizations to define how specific terms, brand names, or industry jargon should be translated (or explicitly not translated), ensuring brand consistency across languages.
- **Profanity Filtering**: Optionally masks or removes profane words from the translated output.
